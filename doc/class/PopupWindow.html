Class: PopupWindow

Defined in: src\popup_window\popup_window.coffee

Overview

-------------------------------------------------------------------------------------------------------------

Popup window widget

this class creates a popup window that has a title and its dragable.

#

@example popup = new PopupWindow(title, x, y)

#

Property Summary

(Integer) popupWidth
popupWidth the width of the popup default 600
(Integer) popupHeight
popupHeight the height of the popup default 400
(Boolean) isVisible
isVisible if popup is visible on current screen default false
(Boolean) allowHorizontalScroll
allowHorizontalScroll if horizontal scrollable default false
(Object) configurations
configurations the configurations about table and savable popup
