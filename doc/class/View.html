Class: View

Defined in: src\screen_manager\View.coffee

Overview

A View is a managed class that contains Javascript, and HTML Template, and a CSS file The base view code is loaded on demand from the server. Something should extend this base view class to implement specific actions.

The base class handles adding the CSS to the document, adding the HTML template to the DOM and setting up variables based on the template.
