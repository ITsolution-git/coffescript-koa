Class: PopupMenuItem

Defined in: src\popup_menu\popup_menu.coffee

Overview

-------------------------------------------------------------------------------------------------------------

popup menu item class to manage single popup menu item

@example

popup = new PopupMenu(title, x, y)

popup.addItem "Item Text", callbackFunction, callbackData

.setBadge(3)

.setIcon('fa fa-home')

.setClass('primary')
